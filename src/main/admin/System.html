System

You can edit System configuration from here, disconnect from the Administrator or even shutdown the Waarp server.

 

Bandwidth Global Limit (B/s):  

The value (in Byte per second) specifies the limit of bandwidth globally (cumulative). 0 means no limit. Must be greater than 1KBs.


CPU Limit (0 to 1):  

CPU Limit as % between 0 and 1. 0 or 1 means no limit.


Connection Limit:  

Connection limit (number of concurrent connections). 0 means no limit.

 

                                                               

 

 

Disconnect from Waarp Administrator     

You can disconnect from this administrator by clicking on this button.

 

 

Shutdown Waarp Server                        

By clicking on this button will result in the Shutdown process of the full Waarp Server (all active transfers will be aborted)..
